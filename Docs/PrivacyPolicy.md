# 隐私政策 (Privacy Policy)

**生效日期：** 2026年2月28日

我们非常重视您的隐私。本隐私政策旨在说明“旅行记账”（以下简称“本应用”）如何处理您的信息。

## 1. 数据存储
本应用是**纯本地应用**。您输入的所有账单数据、拍摄的照片、录制的语音，均**仅存储在您的设备本地**。我们不会将您的任何个人数据上传至任何服务器。

## 2. 数据收集与使用
本应用**不收集、不存储、不分享**任何个人身份信息。
*   **相机权限**：仅用于拍摄购物小票以进行 OCR 识别。识别过程在本地或 Apple 提供的系统级服务中完成。
*   **相册权限**：仅用于从相册选择图片进行识别。
*   **麦克风权限**：仅用于语音记账功能。语音转文字的过程在本地完成。

## 3. 第三方服务
本应用未接入任何第三方统计 SDK 或广告 SDK，不会追踪您的使用行为。

## 4. 数据安全
由于所有数据均存储在您的设备本地，请您务必保管好您的设备。如果您卸载本应用，本地存储的数据将会被一并删除且无法恢复。
